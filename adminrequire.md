A admin side required
First page is Dashboard : Including
Current Datetime
Renting Camera
The number of customer last week
revenue today
Future booking event (next 1 week)
Web visitor count


Second page is Booking Manager : Including
Renting List : A list of camera is in renting now
Future List : A list of camera is in future booking
Past List : A list of camera is in past booking
Bill List : A list of bill
All list above can search by name, phone, id
Export a bill to send for customer as image

Third page is Modify 
Modify camera database : add new one, remove camera, modify price, modify image, modify description, modify category, modify status
Modify customer database : add new one, remove customer, modify name, modify phone, modify email, modify city, modify social, modify status 
Modify booking database : add new one, remove customer, modify name, modify phone, modify email, modify city, modify social, modify status 

Last page is Report
Export report to excel file
